Tentang | Tujuan

Saya Budi Santoso dari Fakultas Teknik & Komputer, Prodi Sistem Informasi, saya bertempat tinggal di Tanjung Batu Kundur. Saya memilih prodi Sistem Informasi karena ingin belajar menggunakan bahasa pemrogaman dan bahasa pemrograman itu ada banyak jenis tetapi yang akan digunakan tidak semua nya, jika ingin belajar semua juga gpp yang jelas saya belajar HTML, CSS, Python, PHP dan JavaScript akan segera di pelajari.

Pada website ini dibuat untuk menempuh tugas mandiri webdesign. Website ini dibuat hanya untuk diliat atau sebagai sample sebuah website dan tidak bisa di akses seperti website lain yang bisa mengirimkan pesan kepada pemilik website. Website ini dibuat dengan kombinasi antara HTML dan CSS serta 1 JavaScript untuk bagian Title bergeraknya. Pembuatan website ini menggunakan style dari Framework Bootstrap dan CSS manual yang dibuat oleh saya sendiri sebagai penambahan style untuk website ini.

Sekian penjelasan tentang website yang dibuat berikut ada beberapa gambar yang saya temukan di browser yaitu pemandangan yang indah. Sekian jika ada salah ketik saya mohon maaf Saya ucapkan Terima Kasih.
